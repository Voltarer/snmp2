## Структура проекта
* **`src/`** — Исходный код агента.
    * `main.c` — Основная логика: опрос портов и обновление модели.
    * `ifTable.c/h` - исходники для реализации табилцы интерфейсов в SNMP-агенте
* **`lib/`** — Библиотеки.
    * `net-snmp` — Стандартный стек протоколов SNMP.
* **`build/`** — Папка с готовым бинарным файлом.

---

## 🛠 Скрипты управления

| Скрипт | Назначение |
| :--- | :--- |
| `./install.sh` | **Генератор модели.** Скрипт выбора ОС, на которой сейчас работаете |
| `./setup.sh` | **Подготовка к сборке** Подготовка общей среды разработки (папки и т.д) |
| `./build_MIPS.sh` | **Подготовка к сборке** Собирает агента под MIPS |
| `./build_UBUNTU.sh` | **Подготовка к сборке** Собирает агента под UBUNTU|
| `./choice.sh` | **Подготовка к сборке** Скрипт выбора ОС, которая требуется |

## Быстрый старт

``` bash
chmod +x install.sh
./install.sh
```

### Готовый бинарный файл будет лежать в build:

    build/my_agent

### Сборка для докера

```sh
BUILDX_GIT_INFO=false docker build -t mips-snmp-agent -f mips_docker/Dockerfile .
```
```sh
docker run -it --name snmp_container -p 1161:161/udp mips-snmp-agent
```
